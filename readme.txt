# XBooks

## Get it work

0. Extract all files, there should be a `xbooks/` folder
1. Install [NodeJs](https://nodejs.org/) **LTS** ver
2. Open cmd/powershell/terminal/bash in `xbooks/`
3. Type `npm install`, press enter/return
4. Type `npm start`, press enter/return
5. Open browser, goto http://localhost:3000
